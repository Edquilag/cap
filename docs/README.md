# ZonalHub Documentation

This documentation set describes the full ZonalHub system for engineering, operations, and security teams.

## Documents

- `docs/SYSTEM_OVERVIEW.md`  
  Product goals, architecture, core modules, and end-to-end request/data flow.

- `docs/API_REFERENCE.md`  
  REST endpoints, query parameters, response schema, and usage examples.

- `docs/DATA_INGESTION_LOGIC.md`  
  BIR dataset fetch pipeline, workbook parsing behavior, canonical mapping rules, and data lineage.

- `docs/PERFORMANCE.md`  
  PostgreSQL search optimization strategy, indexes, query behavior, and tuning checklist.

- `docs/SECURITY.md`  
  Security model, current controls, threat/risk areas, and hardening roadmap.

- `docs/OPERATIONS.md`  
  Setup, migration, deployment workflow, backup/recovery, monitoring, and troubleshooting runbook.

## Reading Order
